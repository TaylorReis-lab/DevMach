## DevMach - Tinder para Desenvolvedores

Plataforma estilo "match" para conectar desenvolvedores com base em tecnologias, projetos e interesses.
Funcionalidades (MVP)

Cadastro de perfil com stack tech
Swipe para match (frontend simulado)
Chat simples após match
Filtros por linguagem/cidade/nível
Backend básico para persistência

# Tecnologias

Frontend: JavaScript / React (ou vanilla + fetch)
Backend: Node.js + Express (se houver server)
Banco: JSON local ou MongoDB (opcional)

# Como rodar
```bash
git clone https://github.com/TaylorReis-lab/DevMach.git
cd DevMach
npm install
npm start
```
